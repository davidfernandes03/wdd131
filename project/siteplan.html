Site Plan - CodeStart Guide

Site Name Purpose Scenarios Colors Typography Wireframe

Site Name

CodeStart Guide: A beginner-friendly resource to start learning programming.

Purpose

This site serves as a simple guide for newcomers to programming, offering foundational knowledge, tips, and resources to help them get started.

Scenarios

  • Which programming language should I start with as a beginner?
  • Where can I find free programming courses?
  • What tools do I need to start coding on my computer?

Colors

  • Primary: Dark Blue (#00509E) - used for headers, buttons, and links.
  • Secondary: Yellow (#FFC107) - for accents, such as borders, icons and navigation bar.
  • Text: Black (#333333) - for body text and paragraphs.
  • Highlight: White (#FFFFFF) - used as the main background color.

Typography

  • Roboto: Used for body text.
  • Poppins: Used for headings and titles.

Wireframe

The wireframe for this site includes a simple layout for both mobile and desktop views:

  • Mobile: Single-column structure with stacked sections.
  • Desktop: Two-column grid with navigation on the left and content on the right.
